# PROG31632 - iOS Development Assignment 1

## Social Media iOS App with Core Data

A comprehensive iOS application demonstrating Core Data integration, programmatic UI development, and social media functionality.

## Project Structure

- **A1_ThanhTungLe/** - Main iOS project with Core Data implementation
- **A1-iOS/** - Additional iOS components
- **Sample/** - Reference implementations and examples

## Key Features

- Facebook login simulation
- Core Data integration for data persistence
- Programmatic UI (no storyboard)
- Post creation and management
- User profile management with image upload/download
- Table view for displaying posts

## Technologies Used

- Swift
- UIKit (Programmatic UI)
- Core Data
- iOS SDK

## Setup Instructions

1. Clone the repository
2. Navigate to `A1_ThanhTungLe/` directory
3. Open `A1_ThanhTungLe.xcodeproj` in Xcode
4. Build and run the project
